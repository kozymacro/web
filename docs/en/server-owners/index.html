Respect for Game Balance

Kozy is designed to enhance players' gaming experience. We design our features to increase player comfort without disrupting game balance or economy. Our goal is to improve the gaming experience, not disrupt game dynamics.

No Technical Interference

Kozy is a 100% pure macro and does not interfere with the game itself. It has no ability to alter game files or mechanics. We do not modify game files, inject code, or develop features that could compromise server integrity or stability.

Our approach is built on respecting both the game's architecture and the work you put into managing your server infrastructure.

Flexible Collaboration

All macro features fundamental to Kozy will remain intact in custom versions. We only customize based on specifically requested restrictions.

Is there anything extra to do for Rise Online?
Select windowed fullscreen from settings. Reset widgets. Increase window transparency. Check the option to lock HP/MP bar and mob HP. (The "Prevent moving hp and target bar" option in Gameplay tab)
  • Set sharpness to 1.
  • Enable "Show Monster HUD".
  • "Show cooldowns and active skill buffs" should be disabled.
  • Keep game and desktop resolutions the same.
  • Don't scale the game. (Don't change interface size. Keep it at 1.) (For low resolutions like 1366, 1.2 is recommended.)
  • Windows scaling should be 100%. It can be 125%, check and change it to 100%.
  • If you're using 2 monitors, play the game on the main screen.
  • When starting for the first time, make sure HP/MP is full.
  • For macros related to creature names, make sure the creature name is in the center and top of the screen.
  • For macros like party genie, make sure to select "Show HP/MP values" in the party list.
  • If you've set up captcha notifications, window transparency must be increased.
  • Graphics cards have settings that cause color changes and may affect some features of the macro. These include vibration, saturation, gamma, color temperature, custom color, sharpness, etc.
  • No other folder, tab or window named Rise should be open.

HP/MP is not working or working incorrectly. What should I do?
Close the macro completely. Open it again. Fill your HP and MP. Without moving the HP and MP indicator, position it against a background without any blue or red tones. For example: don't let it be against the blue sky. Now start the macro.
  • Also, in some PVPs, HP value texts are very large. Setting a percentage that matches these can cause confusion. If you can reduce the text on HP/MP by changing the font, do so.
  • If HP is being used when you're debuffed by Parasite etc., let mana or HP decrease once through normal means. After that, it won't unnecessarily use potions even if you get debuffed by parasite.
  • If you're using it in a virtual PC, make sure to install vmware tools with graphics options selected. For Rise, see the Rise requirements section above.

What should I do to receive notifications via Telegram? (version 2.5.8+)
  1. Type BotFather in Telegram's search. Click on the one with the blue check mark on the right.
  2. Start messaging with BotFather using the button at the bottom of the screen.
  3. An explanation message will come. Click on /newbot in the message.
  4. The message asks what our bot's name will be. This doesn't have to be unique. Choose a name and send it.
  5. Now it asks for username information. The important point is that the username must end with "bot" letters. You may not be able to create a bot on your first try because the username you thought of may already be taken.
  6. If you've successfully completed step 5, you now have a bot. The token given to you will enable communication with the bot. Note this token.
  7. Go to the search window again and type your bot's unique name exactly. Your bot will appear. Select it and start the conversation. Enter the conversation with the /start command.
  8. Enter the token you noted in the notifications window field in Kozy.
  9. If everything is fine, a Test message will be sent to you automatically.
